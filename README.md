🏥 Healthcare Analytics Project

📌 Overview

This project analyzes healthcare data using Python and Power BI. It provides business insights and patient demographic analysis through data visualization and interactive dashboards.

🎯 Project Objectives

- Find the disease with the highest treatment cost.
- Find the department with the highest revenue.
- Compare treatment cost by insurance status.
- Calculate insurance coverage percentage.
- Find the city with the highest average treatment cost.
- Analyze healthcare expenditure by age group.
- Find common diseases by age group.
- Compare treatment cost by gender.
- Identify blood groups with the highest critical cases.

🛠️ Tools Used
