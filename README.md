# 🏥 Healthcare Data Analysis

## Overview

This project analyzes a healthcare dataset using Python to identify trends, perform data cleaning, and generate meaningful insights through exploratory data analysis (EDA).

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Approach

* Loaded and explored the dataset.
* Cleaned missing and duplicate data.
* Performed EDA using statistics and visualizations.
* Generated insights based on healthcare trends.

## Files

* `Healthcare Analysis.ipynb` – Analysis notebook
* `Healthcare_Analysis_Report.pdf` – Final report
* `README.md` – Project documentation
