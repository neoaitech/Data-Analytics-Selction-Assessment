## Project Overview

This project was completed as part of the NeoAI Tech Data Analytics Internship Assessment. The objective was to clean, analyze, and visualize a healthcare dataset using Python and Power BI to generate meaningful business and patient demographic insights.

---

## Objectives

- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Answer business analysis questions
- Analyze patient demographics
- Build an interactive Power BI dashboard
- Present insights through a professional report

---

## Tools & Technologies

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Power BI
- Git & GitHub

---

## Dataset Features

The dataset includes:

- Patient ID
- Age
- Gender
- Disease
- Department
- City
- Treatment Cost
- Insurance Status
- Blood Group
- Outcome
- Admission Date
- Discharge Date

---

## Data Cleaning

The following preprocessing steps were performed:

- Imported the dataset
- Checked missing values
- Removed duplicate records
- Corrected data types
- Converted date columns
- Standardized categorical values
- Created Age_Group column
- Exported cleaned dataset for Power BI

---

## Business Analysis

The following questions were answered:

1. Which disease contributes the highest treatment cost?
2. Which department generates the highest revenue?
3. Average treatment cost by disease and insurance status
4. Percentage of treatment cost covered by insured patients
5. City with the highest average treatment cost

---

## Patient Demographics Analysis

1. Healthcare expenditure by age group
2. Disease distribution across age groups
3. Treatment cost comparison by gender
4. Critical outcomes by blood group

---

The dashboard includes:

- KPI Cards

- Average Treatment Cost by City
- Healthcare Expenditure by Age Group
- Average Treatment Cost by Gender
- Critical Outcomes by Blood Group
- Disease Distribution by Age Group
